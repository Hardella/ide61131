    Installation

    You would require the following to install Hardella IDE:

    • 5 GiB disk space
    • 2 GiB RAM
    • Modern operating system
    • Java 8 (Java SE Development Kit)

    Java 8 setup:

    1. Navigate to jdk8 download
    2. Pick the archive for your operating system
    3. Setup it: next, next, next, …

    Hardella IDE setup:

    1. Navigate to download page
    2. Pick an archive for your operating system
    3. Unzip the archive. Please use the foloder without spaces. For instance, c:\test\Hardella is a good choice, while c:\my documents\Hardella might fail.
    4. You can start Hardella
